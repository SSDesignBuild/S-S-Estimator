Deploy this Supabase Edge Function after uploading the site files:

npx supabase functions deploy send-to-ghl --use-api --no-verify-jwt

Function version included in this package:
v24x-ghl-terms-recursive-descriptions-20260506

Changes:
- Sends a required title while preserving/fetching terms from GHL invoice/estimate settings.
- Fetches invoice settings from both LeadConnector services and backend endpoints.
- Recursively searches the settings/template response for saved estimate terms/conditions fields.
- Leaves customer address inputs out of the app.
- Marks every sent line item taxable and enables automaticTaxesEnabled.
- Replaces generic descriptions with shorter, item/category-specific bullet descriptions.
- Notes that customer supplies light/fan fixtures; S&S installs them.
